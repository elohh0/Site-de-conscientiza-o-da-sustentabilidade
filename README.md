# Site-de-conscientiza-o-da-sustentabilidade

# 🌎 Cuidando do Planeta para o Futuro

## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de conscientizar as pessoas sobre a importância da sustentabilidade e da preservação ambiental. Por meio de informações, ações sustentáveis e atividades interativas, o site busca incentivar hábitos que contribuam para um futuro melhor.

O projeto está alinhado ao tema do Concurso Agrinho:

**"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente".**

## Objetivos

* Incentivar a preservação do meio ambiente.
* Promover a conscientização sobre sustentabilidade.
* Demonstrar a importância da relação entre campo e cidade.
* Apresentar ações simples que podem ajudar o planeta.
* Estimular hábitos sustentáveis no dia a dia.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* imagens geradas pela IA Gemini

## Funcionalidades

* Navegação entre seções do site.
* Informações sobre sustentabilidade.
* Seção sobre o Programa Agrinho.
* Mensagens ambientais interativas.
* Contador de árvores plantadas virtualmente.
* Quiz da Sustentabilidade.
* Modo escuro.
* Layout responsivo para celulares e computadores.
* Efeitos visuais e animações.

## Estrutura do Projeto

* index.html → estrutura do site.
* style.css → estilização e responsividade.
* script.js → funcionalidades e interatividade.

## Orientação

O desenvolvimento do projeto contou com orientação pedagógica da professora Suelen, que auxiliou na organização das atividades e no processo de aprendizagem, assim como as aulas do realizadas dentro do site Alura.com

## Tema Agrinho

O projeto destaca a importância do equilíbrio entre produção e preservação ambiental, mostrando que o desenvolvimento sustentável depende da colaboração entre o campo e a cidade.
